This is a header file that contains MCU specific details, such as:

1. Base addresses of various memories in MCU (Flash, SRAM1, SRAM2, ROM, etc.)

2. Base addresses of various bus domains (AHBx, APBx)

3. Base addresses of various peripherals in different bus domains.

4. Clock management macros

5. IRQ definitions

6. Peripheral Register definition structures

7. Peripheral Register bit definitions

8. Other useful MCU configuration macros
